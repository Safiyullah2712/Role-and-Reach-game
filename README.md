# Roll & Reach Game

## Overview
**Roll & Reach** is a modern, interactive board game built for 2 to 4 players. Inspired by classic games like Snakes and Ladders, this project focuses on a clean, minimal interface with smooth visual feedback and dynamic gameplay elements.

## Design Concept
The game features a distraction-free grid layout that prioritizes clarity and a premium user experience. The design uses a sophisticated color palette that adapts to each player's turn, with subtle animations and micro-interactions—like the rolling dice and shifting turn indicators—that make the board feel alive.

## Features
*   **Dynamic Multiplayer Support**: Play with 2, 3, or 4 players with custom naming options.
*   **Snakes & Ladders Logic**: Strategic special tiles that can either boost your progress or send you back.
*   **Move History Tracking**: Each player has an expandable "Move History" card that logs their dice rolls as visual icons.
*   **Visual Turn Indicators**: The dice container dynamically moves and changes color to highlight the active player.
*   **Responsive Welcome Experience**: A dedicated setup screen to configure players before the race begins.
*   **Interactive Dice**: Clickable, animated dice with realistic rolling effects.

## Page Structure
The game is organized into several key components:
*   **Welcome Page**: A centralized hub for selecting player count and entering participant names.
*   **Game Board**: A 10x10 grid where player tokens (custom icons) move in real-time.
*   **Player Sidebars**: Left and Right containers that house player-specific information and move history.
*   **Victory Modal**: A celebratory overlay that appears when a player reaches the 100th cell.

## User Experience
As players interact with the game:
1.  **Selection**: Users choose their player count and names on a clean welcome screen.
2.  **Turn-Based Action**: The active player's name is highlighted, and the dice moves to their section for a convenient rolling experience.
3.  **Progression**: Tokens move across cells, reacting to snakes and ladders with instant status updates.
4.  **History Insight**: Players can toggle their "Move History" to track past luck and strategy throughout the game.

## Technologies Used
*   **HTML**: For semantic structure and layout.
*   **CSS**: Custom styling, animations, and a responsive flexbox-based design.
*   **JavaScript**: For core game logic, state management, and real-time UI updates.

## Purpose
The purpose of this project is to provide a clear understanding of fundamental game logic, state management, and real-time UI updates, serving as a practical example for understanding modern web development principles.
